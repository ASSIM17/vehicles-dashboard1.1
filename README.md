# Vehicle Sales Dashboard

Esta aplicación web fue desarrollada con **Streamlit** y permite explorar un conjunto de datos
de anuncios de venta de coches mediante visualizaciones interactivas.

El proyecto forma parte del **Sprint 7** y tiene como objetivo practicar el flujo completo
de desarrollo: análisis exploratorio, creación de una app web y despliegue en la nube.

---

## Funcionalidades
- Encabezado descriptivo en la aplicación
- Vista previa del dataset
- Histograma interactivo (Plotly)
- Gráfico de dispersión interactivo (Plotly)
- Interacción mediante botones (Streamlit)

---

## Dataset
El proyecto utiliza el archivo:

- `vehicles_us.csv`

Este archivo contiene información sobre anuncios de venta de coches, como:
precio, kilometraje (odometer), año del vehículo y otras características.
